▶ Как получить токен доступа?

1. Открой эту ссылку на GitHub

2. Укажи описание (например, Nik CMS)

3. Срок действия (Expiration): желательно No expiration

4. Галочка: ✅ repo (полный доступ к коммитам)

⚠️ ВАЖНО: У Classic токена обязательно должна быть включена галочка "repo", а у Fine-grained токена �� права "Contents: Read/Write". Иначе вы не сможете опубликовать изменения.

5. Нажми Generate token и скопируй токен сюда

Токен сохраняется исключительно в вашем браузере (localStorage) и не улетает на сторонние сервера.
